# ClairDevis Auto - Démo GitHub

Ce dossier est prévu pour une mise en ligne simple sur GitHub Pages afin de montrer
un exemple visuel à des collègues.

## À uploader

Uploader tout le contenu du dossier :

- `index.html`
- `styles.css`
- `app.js`
- `assets/hero-devis-auto.png`
- `README.md`

## Important

Cette version est volontairement statique :

- aucun paiement réel ;
- aucune transmission de formulaire ;
- aucun stockage de devis ;
- aucun serveur Node nécessaire.

Le bouton `Simuler le dépôt` ouvre simplement une fenêtre de démonstration.

Pour une vraie version exploitable, il faudra reprendre le dossier
`devis-auto-platform`, ajouter un hébergement avec backend, connecter Stripe et
prévoir un stockage sécurisé des devis.
